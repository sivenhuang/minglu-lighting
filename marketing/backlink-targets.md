## Execution workflow (repeat per platform)

1. Register with **company email** (not personal Gmail if a business domain exists).
2. Paste **exact NAP** + choose the right-length **boilerplate**.
3. Add website URL, upload 3–5 product images (see GBP image checklist).
4. Link existing Facebook/Instagram; add new socials as created.
5. Log the profile URL + login in a tracker (see below).
